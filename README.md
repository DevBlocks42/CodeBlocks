# CodeBlocks

Casse brique en 2 dimensions codé en C++ à l'aide de la bibliothèque SFML (https://www.sfml-dev.org/index-fr.php).

# Installation

- Linux :

    Prérequis : G++; SFML

    `cd CodeBlocks/`

    `sh Binaries/Linux/compile.sh`

    `./CodeBlocks`
